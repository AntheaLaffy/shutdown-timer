モダンでカスタマイズ可能なデスクトップシャットダウンタイマー。カウントダウンまたは時間指定でシステムshutdownをスケジュールでき、ビジュアルカスタマイズ、多言語サポート、プロフェッショナルなUI/UXを備えています。

> ✨ **大幅アップデート**: コンパクトな縦型レイアウトから、最新のサイドバー型アプリケーションへと生まれ変わりました。

## ✨ 機能

### 🎯 コアタイマー機能

- **カウントダウン方式**: 時/分/秒でshutdownまでの時間を設定
- **時間指定方式**: 特定の時刻にshutdownをスケジュール
- **着信音サポート**: shutdown前のカスタム音声アラート
- **システム連携**: スリープ防止、トレイ最小化、自動起動
- **安全機能**: いつでもキャンセル可能、警告通知

### 🎨 高度なテーマシステム

- **10以上のプリセットテーマ**:
  - Cyberpunk（サイバーパンク霓虹）
  - Fresh（ミントグリーン）
  - Ocean Blue、Forest Green、Sunset Orange
  - Elegant Purple、Midnight Dark、Solarized Light
  - Monochrome、Retro 80s
- **完全なカスタマイズ**: 無制限のカスタムテーマを作成・保存可能
- **リアルタイムプレビュー**: カスタマイズ中国家康 изменения мгновенно
- **インポート/エクスポート**: JSONファイルでテーマを共有

### 🖌️ ビジュアルカスタマイズ

- **カラーシステム**: 12以上の意味論的カラー変数
- **フォント選択**: 50以上の厳選Google Fontsから選択
- **レイアウト制御**: 間隔、ボーダー半径、影を微調整
- **背景画像**: カスタム背景をアップロードして透明度を調整
- **タイポグラフィ**: 見出し、本文、等幅、アクセント用に別々のフォント

### 🌐 国際化

- **3つの言語**: 简体中文、英語、日本語
- **動的切り替え**: 再起動なしで言語を変更
- **完全なUI翻訳**: すべてのインターフェース要素を翻訳済み

### 🧭 モダンなUI/UX

- **サイドナビゲーション**: 整理された3ページレイアウト
- **スムーズアニメーション**: ページ間のフェード/スライドトランジション
- **レスポンシブデザイン**: デスクトップ最適化、モバイル対応
- **アクセシビリティ**: WCAG準拠、キーボードナビゲーション対応

## 🚀 クイックスタート

### 必要環境

- [Node.js](https://nodejs.org/) 18+ と npm
- [Rust](https://www.rust-lang.org/)
- [Tauri CLI](https://tauri.app/v2/guides/getting-started/prerequisites)

### インストール

```bash
# リポジトリをクローン
git clone <リポジトリURL>
cd shutdown-timer

# フロントエンド依存関係をインストール
cd src
npm install

# Tauri依存関係をインストール
cd ../src-tauri
cargo build
```

### 開発モード

```bash
# srcディレクトリから
npm run dev

# またはプロジェクトルートからTauriで
npm run tauri dev
```

### ビルド

```bash
# 開発ビルド
npm run build

# 本番ビルド（Tauri）
npm run tauri build
```

## 📖 使い方

### タイマーを設定する

1. **アプリケーションを開く**
2. **サイドバーから「タイマー」ページを選択**
3. **方式を選択**:
   - **カウントダウン**: 時/分/秒を入力
   - **時間指定**: 時刻を入力（24時間制）
4. **オプション**: アラート用の着信音ファイルを選択
5. **設定を調整**:
   - カウントダウン中にスリープを防止
   - システムトレイに最小化
   - システム起動時に自動起動
6. **「カウントダウン開始」または「スケジュール設定」をクリック**

### 外観をカスタマイズする

1. **サイドバーから「外観」ページに移動**
2. **各タブを探索**:
   - **色**: すべてのテーマ色を調整
   - **フォント**: テキストタイプ別にフォントを選択
   - **レイアウト**: 間隔、ボーダー、影を微調整
   - **背景**: 画像をアップロードして透明度を調整
   - **テーマ**: プリセットを閲覧してカスタムテーマを管理
3. **カスタムテーマを保存**して後日使用
4. **テーマをインポート/エクスポート**して共有

## 🎨 テーマシステム

### プリセットテーマ

アプリケーションには10のプロフェッショナルデザイン済みテーマが付属:

| テーマ | 説明 | 分類 |
|--------|------|------|
| **Fresh** | クリーンで柔らかい緑のアクセント | ライト |
| **Cyberpunk** | ネオンシアン/マゼンタの未来派ダークテーマ | ダーク/ネオン |
| **Ocean Blue** | 穏やかな海洋カラー | プロフェッショナル |
| **Forest Green** | 自然でアースカラーのトーン | ナチュラル |
| **Sunset Orange** | 温かい夕日インスパイアパレット | ウォーム |
| **Elegant Purple** | 洗練された気品ある紫テーマ | エレガント |
| **Midnight Dark** | アクセント付きプライバシーモード | ダーク |
| **Solarized Light** | プロフェッショナルな可読性最適化 | ライト/プロ |
| **Monochrome** | グレースケールと単一アクセントカラー | ミニマル |
| **Retro 80s** | ヴィクトリアンなRetro配色 | Retro |

### カスタムテーマの作成

1. **プリセットまたは現在のカスタマイズから開始**
2. **色、フォント、レイアウトを調整**
3. **テーマタブの「テーマを保存」をクリック**
4. **テーマ名前を付けてオプションの説明を追加**
5. **JSONとしてエクスポート**して共有
